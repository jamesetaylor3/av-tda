### datasets folder

The directories of each of the datasets go here. They are gitignored due to their size.

Store them in the director as so.

```
prediction-dataset/
  +- scenes/
        +- sample.zarr
        +- train.zarr
        +- train_full.zarr
  +- aerial_map/
        +- aerial_map.png
  +- semantic_map/
        +- semantic_map.pb
  +- meta.json
```

You can download them [here](https://self-driving.lyft.com/level5/download/)
